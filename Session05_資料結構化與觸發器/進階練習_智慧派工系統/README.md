# 進階練習：🤖 智慧任務派工與自動提醒系統

> 對應 Session 5 ｜ 辦公室自動化 AI 主題

## 🎯 練習重點
- **資料結構化**：將試算表二維陣列轉為更易讀的物件陣列 (Object Array)。
- **智慧派工邏輯**：
  1. **同部門匹配**：任務自動指派給需求部門內的人員。
  2. **最少負擔優先**：自動尋找部門內「目前任務數」最少且未滿載的員工。
- **效能優化 (批次寫回)**：在記憶體中更新二維陣列，最後一次性 `setValues` 寫回，大幅減少 API 呼叫次數。
- **自動偵測與提醒**：每日檢查並標記逾期、即將到期的任務。

## 💼 情境
公司有多項待辦任務，主管需手動分配。利用 GAS 建立智慧派工系統，自動依據任務需求部門及每位員工的目前工作量分配任務，並在任務逾期前發出提醒。如果該部門人力皆滿載，系統也會自動發出無可用人力的警報。

## 🧠 進階挑戰
1. **技能匹配**：除了部門外，根據任務類型進一步匹配對應技能的員工。
2. **加班預警**：當員工任務數超過最大承載時自動發出預警通知。
3. **Email 隔日提醒**：整合 `MailApp` 或 `GmailApp`，將逾期任務名單自動發信通知負責人。
